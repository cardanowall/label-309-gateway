Mint operator token

Exchanges the operator ROOT credential for a short-lived operator token (24 h by default) — the token this console should run on. The root secret authorizes only this single call, stays in this form field, and is never stored.

Register wallet

Root-gated: registration binds a shared-keyring Cardano signing key to this operator, so it requires the operator ROOT credential — an ordinary operator token answers 403. The address must be one this server's keyring physically holds (it is verified against the signing key at unlock); an address the keyring cannot sign for is refused.
